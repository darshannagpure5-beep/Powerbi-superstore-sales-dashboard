# 📊 Super Store Sales Dashboard (Power BI)

##   Project Overview
This project presents an interactive **Power BI dashboard** built using the Superstore dataset to analyze sales performance, profits, shipping modes, and regional trends.

The dashboard helps businesses understand:
- Sales distribution
- Profit trends
- Customer segments
- Regional performance

##   Key Insights

✔ Consumer segment contributes the highest sales  
✔ Standard Class shipping mode dominates orders  
✔ Office Supplies category generates highest revenue  
✔ COD is the most used payment method
